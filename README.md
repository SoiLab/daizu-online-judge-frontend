## How to run 

### develop

Run frontend.
```sh
$ npm install
$ npm run start
```

Run fake api.
```sh
# If pipenv is not installed.
# $ pip install pipenv

$ cd fake_api
$ pipenv install
$ pipenv run python main.py
```
